Задания бывают 3-х видов: разовые, индивидуальные, локационные.

Разовое задание выполняется всеми участниками только 1 раз. Экономится время на поиск исполнителей, чтобы не приходилось несколько раз выполнять одинаковую работу. Пример: организация мероприятия. 

Индивидуальное задание могут выполнять все желающие. Можно заранее набирать нужное количество участников, после чего участвуют все или никто. Пример: участие в уличном мероприятии.

Локационное задание предлагает простые задание для разных локаций. Нужные локации заранее бронируются, чтобы гибко распределять работу между всеми желающими. Пример: расклейка объявлений.
